## Requisitos

* Java SDK 8+
* IntelliJ or Eclipse
* Maven 3
* Git


## Instalação


```bash
git clone git@github.com:acelera-dev/acelera-dev-brasil-2019-01.git
cd acelera-dev-brasil-2019-01
mvn clean install
```
